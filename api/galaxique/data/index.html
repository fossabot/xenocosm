    This is the API documentation for Xenocosm a procedural space trader.

    This is the API documentation for Xenocosm a procedural space trader.

    The game's codebase is divided into a number of modules.

    • pseudoglot: logic and data types for procedural phonologies
    • galaxique: logic and data types for the procedural multiverse
    • economancy: logic and data types for the procedural markets
    • xenocosm: logic and data types combining the above into a space trader
    • xenocosm.http: An HTTP application of the game via http4s

    Please refer to the documentation for a more details about the application.
